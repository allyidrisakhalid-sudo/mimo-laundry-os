# Auth Copy EN/SW  Mimo Phase 2

## 1) Auth Copy Principles

- EN must be premium, short, clear
- SW must be natural and clean
- no robotic translation
- no contradictory labels
- all copy must support i18n parity
- auth copy must align with the public-site and brand tone from previous chapters
