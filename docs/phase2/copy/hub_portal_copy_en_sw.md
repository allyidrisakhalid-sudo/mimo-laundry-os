# Hub Portal Copy EN/SW  Mimo Phase 2

## 1) Hub Portal Copy Principles

- EN must be short, calm, operationally clear
- SW must be natural and workplace-usable
- no robotic translation
- no admin jargon
- no decorative marketing phrasing inside operations screens
- every line must support fast staff action

## 6) Hub Portal Copy Lock Statement

The hub portal copy is locked as a short, calm, implementation-ready EN/SW baseline for tablet and web operations. Copy stays action-oriented for intake, processing, QC, dispatch, and key system states so hub staff can move through daily work quickly without confusion or admin jargon.
