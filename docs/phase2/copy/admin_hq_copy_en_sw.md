# Admin HQ Copy EN/SW  Mimo Phase 2

## 1) Admin HQ Copy Principles

Locked principles:
- EN must be short, calm, executive-clear
- SW must be natural and business-usable
- no robotic translation
- no unnecessary finance jargon
- no decorative marketing language in admin HQ
- every line must support daily admin control and confidence
