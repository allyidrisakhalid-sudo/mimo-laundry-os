# Help Entry Points Spec  Mimo Phase 2

## 1) Help Entry Principles

Locked help entry principles:
- every role must have one obvious help path
- help must be contextual, not hidden
- help must not fragment into random page-level duplication
- help entry must point to the right support destination
- help must reduce hesitation, not interrupt work
- contextual help beats training manuals
- support wording must stay calm and short

## 2) Customer Help Entry Points

- profile/settings entry to help
- order detail support/help entry
- public /help route remains valid
- WhatsApp/support route available where appropriate
- no duplicate giant help screen inside customer home

## 3) Driver Help Entry Points

- profile/help entry
- task-detail help/support entry where proof or issue confusion may occur
- no separate training module in P2.6

## 4) Hub Help Entry Points

- dashboard/help entry
- order-detail contextual help for exceptions/process clarification
- no SOP duplication inside every queue page

## 5) Affiliate Help Entry Points

- dashboard/help entry
- order-detail contextual help
- finance/help entry for affiliate admin where payout clarity matters
- public /help route remains secondary, not primary, for portal users

## 6) Admin and Dev Help Entry Points

- concise support/help access from profile/account area
- contextual help around sensitive controls where needed
- no cluttered documentation center introduced in P2.6

## 7) Help Surface Rules

- help entry label must be plain
- help destination must be deterministic
- contextual help can be inline or linked, but must not overwhelm the page
- portal help and public help must not contradict each other
- help entry must align with role permissions and real support paths

## 8) Help Entry Lock Statement

The Phase 2 help entry model is locked to a contextual, role-correct system with one obvious support path per role. Help is discoverable from the right portal surfaces, remains calm and concise, and routes users to real support destinations without duplicating a training center or fragmenting guidance across redundant screens.
