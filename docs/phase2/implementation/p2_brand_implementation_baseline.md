# P2 Brand Implementation Baseline  Mimo Phase 2

## 1) Purpose

This file converts the approved P2.1 brand system into the implementation control layer for all Phase 2 UI work.

## 2) Locked Brand Direction

- Midnight Silk
- premium smooth operational luxury
- deep midnight dark anchors
- silk-white reading surfaces
- restrained teal/blue/gold accents
- rounded geometry
- calm, premium, trustworthy, operationally sharp

## 3) Locked Token Source

- all future UI must pull color, typography, spacing, radius, shadow, border, and motion choices from the approved shared token source
- no local page token invention allowed

## 4) Locked Font Source

- heading font: Plus Jakarta Sans
- body font: Inter
- no extra decorative families

## 5) Locked Icon Source

- Lucide only
- rounded outline style only
- no mixed icon families

## 6) Locked Imagery Direction

- real, premium, believable
- no cheesy stock look
- no cartoon mascot direction
- imagery remains subordinate to usability

## 7) Implementation Guardrails

- no random colors
- no random radii
- no local shadows outside the system
- no page-level font experiments
- no alternate icon packs
- no unapproved illustration style drift

## 8) Downstream Dependency Rule

- all component and screen implementation after P2.1 must consume this brand layer instead of inventing local styling

## 9) Baseline Lock Statement

- The Phase 2 brand implementation baseline is now the single source of visual truth for product implementation.
