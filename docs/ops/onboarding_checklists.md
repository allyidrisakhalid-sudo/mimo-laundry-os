# Onboarding Checklists

## Zone

- [ ] Zone name created
- [ ] Boundaries / metadata recorded
- [ ] Zone marked active
- [ ] Zone visible in admin list
- [ ] Zone visible in ordering UI

## Hub

- [ ] Hub created
- [ ] Hub linked to zone
- [ ] Address label set
- [ ] Capacity fields set
- [ ] Supported tiers set
- [ ] Hub marked active
- [ ] Hub staff accounts created
- [ ] Test order routes to correct hub
- [ ] Hub staff sees only hub orders

## Affiliate

- [ ] Affiliate shop created
- [ ] Affiliate linked to zone
- [ ] Commission plan assigned
- [ ] Affiliate marked active
- [ ] Affiliate staff accounts created
- [ ] Shop appears only in correct zone selection
- [ ] Cross-shop access returns 403

## Driver

- [ ] Driver user created
- [ ] Driver profile created
- [ ] Home zone set
- [ ] Availability set to AVAILABLE
- [ ] Driver marked active
- [ ] Admin trip assignment works in-zone
- [ ] Driver sees assigned tasks only
- [ ] Cross-zone assignment rejected

## Mini validation pack

- [ ] Door order created in new zone
- [ ] Shop order created in new zone
- [ ] Hybrid return order created or explicitly skipped
- [ ] Hub assignment correct
- [ ] Shop filtering correct
- [ ] Driver task visibility correct
- [ ] Timeline updates present
- [ ] Invoice / receipt generated
- [ ] Commission ledger entry exists for shop delivered+paid
- [ ] Daily close / zone report reflects activity
